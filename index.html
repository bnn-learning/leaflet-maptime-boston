Exercise: Create a Leaflet Map

This Leaflet map is based on Maptime Boston's Leaflet Tutorial written by Andy Woodruff, Ryan Mullins and Cristen Jones
Maptime Boston's Leaflet Tutorial

I have modified the code to work with Leaflet 1.7.1

  • Leaflet bug: height: 100% does not work on the map div
  • Workaround: add style html, body {height: 100%} in the head, so div's inherit 100%
  • Leaflet issue: default marker icons are not found locally
  • Fix: load Leaflet scripts and css from unpkg.com
